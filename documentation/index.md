# Ngx Design Documentation

This folder explains how to work with the Design catalog source repository and its organization override workflow.

Ngx Design is a static Angular design catalog. It is used to preview reusable website directions and implementation briefs. Keep it simpler than a delivered business website: static routes, local assets, and clear design examples.

## Documentation Map

- `org-branches.md`: how CI/CD organization override branches are structured and updated
- `release-checklist.md`: checks before publishing master or preparing org updates

## Agent Guidance Contract

Root `AGENTS.md` remains delivery-facing. Use this `documentation/` folder for source-maintainer workflow details that should not be copied into delivered org repositories.
